# Writeups

This folder stores reusable writeup structure and future project documentation templates.

## Available Template

- [writeup-template.md](writeup-template.md): A starter format for documenting labs, analysis projects, and cybersecurity learning exercises.

## Suggested Format

Most portfolio writeups should include:

- Objective
- Tools used
- Steps taken
- Findings
- Security impact
- Recommendation
- Reflection
